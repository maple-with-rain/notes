§1复变积分

定义

设C是复平面上的曲线，函数f(z)在C上由定义。将曲线C分割为n段，分点为z0=A,z1,z2,...,zn=B，ζ是zk−1→zk段上的任意一点，作和数

∑k=1nf(ζk)(zk−zk−1)≡∑k=1nf(ζk)Δzk

若当n→∞，使得max|Δzk|→0时，此和数的极限存在，且与ζk的选取无关，则称此极限值为函数f(z)沿曲线C的积分，记为

∫Cf(z)dz=limmax|Δzk|→0∑k=1nf(ζk)Δzk

  • 复变积分实际上是两个实变线积分的有序组合

    ∫Cf(z)dz=∫C(u+iv)(dx+idy)=∫C(udx−vdy)+i∫C(vdx+udy)

  • 如果C是分段光滑曲线，f(z)是C上的连续函数，则复变积分一定存在

基本性质

  1. 若积分∫Cf1(z)dz,∫Cf2(z)dz,...,∫Cfn(z)dz均存在，则：

    ∫C[f1(z)+f2(z)+⋯+fn(z)]dz=∫Cf1(z)dz+∫Cf2(z)dz+⋯+∫Cfn(z)dz

  2. 若C=C1+C2+⋯+Cn，则：

    ∫C1f(z)dz+∫C2f(z)dz+⋯+∫Cnf(z)dz=∫Cf(z)dz

  3. ∫C−f(z)dz=−∫Cf(z)dz，C−为C的逆向

  4. ∫Caf(z)dz=a∫Cf(z)dz，a为常数

  5. |∫Cf(z)dz|⩽∫C|f(z)||dz|

    proof.

    ∫Cf(z)dz=limmax|Δzk|→0∑k=1nf(ζk)Δzk=limmax|Δzk|→0∑k=1n|f(ζk)Δzk|

    由此可导出：

    |∫Cf(z)dz|⩽Ml，其中M为|f(z)|在C上的上界，l为C的长度

§2Cauchy定理

Cauchy定理

如果函数f(z)在区域G―中解析，则沿G―的边界C有：

∮Cf(z)dz=0

为简单起见，先默认定理f′(z)在G―中连续

由格林公式：

∮Cf(z)dz=∮C[udx−vdy]+i∮C[vdx+udy]=∬G―[−∂v∂x−∂u∂y]+i∬G―[∂u∂x−∂v∂y]=0

对单连通域，C就是一个简单的有向曲线，对复连通域，可通过增添路径变为单连通域，得到相似结论

∮C0f(z)dz+∑in∮Ci−f(z)dz=0

注：

  • 此处的区域只能是一个有界区域，不能是(绕∞点的)无界区域

  • 即使f(z)在∞点解析，它绕∞点一周的积分也可以并不为0

  • Cauchy−Riemann方程与Cauchy定理描述的东西有一定相似性🤔

∮C(z−a)ndz={2πi,n=−1，且C内包含a点0，其它n∈Z的情况

解析函数的不定积分

对于单连通区域，固定起点z0令终点z为变点，则作为积分上限的函数

∫z0zf(z)dz=F(z)

是单连通区域G内的单值函数，称为f(z)的不定积分（基于单连通域的单值性）

如果函数f(z)在单连通区域G内解析，则F(z)=∫z0zf(z)dz也在G内解析，且

F′(z)=ddz∫z0zf(z)dz=f(z)

具体证明可由F(z)与F′(z)的定义以及f(z)的连续性得出

若Φ′(z)=f(z)，则Φ(z)称为f(z)的原函数

可用于求解积分：

∫abf(z)dz=Φ(b)−Φ(a)

§3两个重要引理

小圆弧引理

若函数f(z)在z=a点的(空心)邻域内连续，且当θ1≤arg⁡(z−a)≤θ2，且|z−a|→0时，(z−a)f(z)⇒k((z−a)f(z)一致趋于k)，则

limδ→0∫Cδf(z)dz=ik(θ1−θ2)

其中Cδ是以z=a为圆心，δ为半径，夹角为θ2−θ1的圆弧

proof.

易得：

∫Cδdzz−a=i(θ2−θ1)

而：

|∫Cδf(z)dz−ik(θ2−θ1)|=|∫Cδ(f(z)−kz−adz)|≤∫Cδ(|(z−a)f(z)−k||dz||z−a|)

当δ→0时，

|(z−a)f(z)|→0|dz||z−a|=iδeiθdθδeiθ=idθ

由此可得

|∫Cδf(z)dz−ik(θ2−θ1)|→0

大圆弧引理

若函数f(z)在z=∞点的(实心)邻域内连续(此处结合后续条件即f(z)→0🤔这0还你妈有方向就很神奇)，且当θ1≤arg⁡(z−a)≤θ2，且z→∞时，zf(z)⇒K(zf(z)一致趋于K)，则

limR→0∫CRf(z)dz=iK(θ1−θ2)

其中CR是以z=0为圆心，R为半径，夹角为θ2−θ1的圆弧

证明过程与小圆弧引理相似

§4Cauchy积分公式

有界区域的 Cauchy 积分公式

设 f(z) 是区域 G― 中的单值解析函数，G― 的边界 C 是分段光滑曲线，a 为 G 内一点，则

f(a)=12πi∮Cf(z)z−adz

其中积分路线沿 C 的正向

根据Cauchy积分公式可以将区域中含非解析区域的函数化为

特殊形式

取C为以a为圆心，R为半径的圆周，z−a=Reiθ

dz=Reiθidθf(a)=12π∫02πf(a+Reiθ)dθ

由此得出均值定理：

解析函数f(z)在解析区域G内任意一点a的函数值f(a),等于 (完全位于G内的)以a为圆心 的任一圆周上的函数值的平均

无界区域的Cauchy积分公式

如果 f(z) 在简单闭合围道 C 上及 C 外解析，且当 z→∞ 时，f(z)→0，则 Cauchy 积分公式

f(a)=12πi∮Cf(z)z−adz

仍然成立，此处 a 为 C 外一点，积分路线 C 为顺时针方向

proof.

对于无界区域，需要假设 f(z) 在简单闭合围道 C 上及 C 外 (包括无穷远点) 单值解析. 类似地，现在计算

12πi∮Cf(z)z−adz

其中 a 为 C 外一点，积分路线 C 的走向是顺时针方向，即绕无穷远点的正向

在C外再作一个以原点为圆心，R为半径的大圆CR，

这样，对于C和CR所包围的复连通区域，根据有界区域的 Cauchy 积分公式，有：

12πi∮CRf(z)z−adz+12πi∮Cf(z)z−adz=f(a)

当R→∞时，显然CR必然能够包括C和a，此时，由大圆弧引理

limR→∞12πi∮CRf(z)z−adz=f(∞)

因此，当f(∞)⇒0时，可以得到无界区域的Cauchy积分公式

§5高阶导数公式

从Cauchy积分公式，可以推导出高阶导数公式：

若f(z)在G―中解析(这保证了Cauchy积分公式的使用)，则在G内f(z)的任何阶导数f(n)(z)均存在，且

f(n)(z)=n!2πi∮Cf(ζ)(ζ−z)n+1dζ

其中C是G―的正向边界，z∈G

proof.

对于f′(z)

f′(z)=limh→0f(z+h)−f(z)h=limh→012πi⋅1h(∮Cf(ζ)ζ−z−hdζ−∮Cf(ζ)ζ−zdζ)=limh→012πi(∮Cf(ζ)(ζ−z−h)(ζ−z)dζ)

limh→012πi(∮Cf(ζ)(ζ−z−h)(ζ−z)dζ)=12πi∮Cf(ζ)(ζ−z)2dζ
